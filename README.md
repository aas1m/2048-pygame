# 2048-pygame
A pygame implementation of the popular single-player puzzle game, 2048.

## Getting Started
1. Get Python 3.x and clone this repository
2. Get [pip](https://www.makeuseof.com/tag/install-pip-for-python/), then install pygame:\
    ```$ pip install pygame```

3. Run the game:\
    ```$ python main.py```
